## Setup
1. Abrimos el proyecto (preferible en VSCode) 
2. Localizamos la carpeta \src\database
3. En el gestor de Mysql importamos el archivo db.sql
4. npm install
5. npm run dev
6. En un navegador colocamos localhost:3000
